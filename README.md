## CST-305-Project7

### Requirements:
- Python 3.x
- NumPy
- Matplotlib

### Running the Program:
1. Open a terminal.
2. Navigate to the directory containing `main.py`.
3. Enable the virtual environment:
   ``` 
   source /.venv/bin/activate 
   ```
4. Run the program:
   ```
   python3 main.py
   ```
